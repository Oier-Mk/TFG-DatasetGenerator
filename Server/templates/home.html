Stable Diffusion para Fine tuning

Bienvenido a la página de Fine Tuning de Stable Diffusion. En esta página se puede generar un dataset a partir de una serie de imagenes y un modelo preentrenado. Se personaliza el modelo con las imágenes personalizadas y se puede usar dicho modelo para inferir datasets de imágenes personalizadas.

  • Introducción

    Los desarrolladores de Stable Diffusion han desarrollado una función llamada Fine Tuning que permite personalizar un modelo preentrenado mediante el entrenamiento con una pequeña cantidad de datos etiquetados. Esto implica reentrenar el modelo original de Stable Diffusion con los datos deseados, utilizando dos métodos de entrenamiento: interno y externo.

  • Tipos de entrenamiento

    Existen dos tipos de entrenamiento en este tipo personalización, el entrenamiento externo y el interno. Abajo se pueden apreciar las diferencias entre ambos, el interno realiza cambios en la imagen original cambiando únicamente el fondo, mientras que el externo realiza cambios en la imagen original cambiando el objeto y manteniendo el fondo.

  • Ejemplo de uso

    Teknei quiere desarrollar un sistema de control de calidad automático para desechar galletas rotas en una empresa de panadería, pero el cliente solo proporciona 20 imágenes de galletas rotas. Por ello, tiene que buscar otras formas de conseguir más imágenes para entrenar su modelo de IA supervisado.

    Fragmento de dataset original

    Para este tipo de problemáticas, es preciso realizar un entrenamiento con las imágenes proporcionadas por el cliente y los diferentes valores para el Learning Rate y Training Steps. Entre estos parámetros, se requiere un equilibrio entre los parámetros para un buen entrenamiento, generando una situación semejante a la siguiente gráfica de la loss fuction.

    Grafica relacion Learning Rate y Training Step.

    En caso de ser asi, encontrar un valor de Learning Rate y Training Steps en el mínimo de la función de pérdida, es decir, para generar una mayor precisión o menor tasa de error en el modelo.

    Resultados del entrenamiento con los diferentes Training Steps y Learning Rate.

    En este caso, con un dataset de imágenes reducidas podemos considerar que un Learning Rate de 4E-5 y un Training Steps de 900, se obtiene un modelo con una precisión bastante buena en diferentes dataset de esos tamaños.

  • Tipos de inferencia

    Un scheduler es un componente de la inferencia de Stable Difusión que define el proceso de eliminación de ruido para un modelo de difusión. Estos algoritmos generan de manera diferente la imagen de salida, por lo que es necesario estudiar cuáles son aquellos con mejores resultados.

    Diferentes schedulers con el dataset de entrada.

    En este caso, con un dataset de imágenes reducidas podemos considerar que con un Scheduler PNDM, se obtiene un conjunto de imágenes con una precisión bastante buena con un dataset ese tamaño.
